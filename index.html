Type of Units in Css

  1. Absolute Units (fixed):
    • px(pixels)
    • in(inches)
    • cm(centimeters)
    • nm(milimeters)
    • pt(points)
    • pc(picas)
  2. Relative Unite:
    • em()
    • rem(root em)
    • %(percentage)
    • vw(viewport Width)
    • vh(viewport Hight)
